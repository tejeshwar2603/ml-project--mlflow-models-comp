Forecast vs actual

Residuals (predicted - actual)

Random scatter near zero is healthy. A wave pattern in the residuals means the model is missing a seasonal cycle (the classic ARIMA failure mode).

Error histogram

Box plot

Narrow distribution = consistent model. Right-skewed = the model systematically misses spikes - dangerous for under-provisioning detection.

Horizon degradation (walk-forward MAPE)

How fast error grows as the forecast reaches further into the future - critical for how much lead time a model actually gives capacity planning.

Dataset size impact / learning curves

Robustness to missing data

ARIMA/SARIMA are fed raw gaps directly (statsmodels missing="drop"). XGBoost goes through this repo's existing imputation pipeline (interpolate/ffill/bfill in features.py) before training.

Model tradeoff radar

Business decision metrics

Critical miss: model said "hold" but the server actually breached 85% - the dangerous under-provisioning case. False alarm: model said "scale" but it wasn't needed.
